# plasticFlower — MVP Concept

---

## What It Is

A live web application that:

1. **Listens** to a talk/lecture via microphone
2. **Transcribes** speech in real-time
3. **Extracts** concepts and relationships using an LLM
4. **Visualises** an emergent knowledge graph that grows as the talk progresses
5. **Organises** related concepts into "Flowers" (thematic clusters)

---

## Key Differentiator

**No predefined schema.** The LLM discovers what matters during the talk.

---

## Name

**plasticFlower** — knowledge that blooms and reshapes as understanding grows.

---

## Target Use Case

Single user capturing a live talk, lecture, or presentation. Not meetings with multiple speakers.
